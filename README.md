## نکته مهم Yoast درباره تصویر

اگر Yoast می‌نویسد «تصاویر: هیچ تصویری در این صفحه ظاهر نمی‌شود»، این با متن حل نمی‌شود. باید در ووکامرس تصویر شاخص/گالری محصول را آپلود کنی. این ابزار فقط alt آماده و سئوشده می‌دهد.

## نکته مهم درباره لینک خروجی
