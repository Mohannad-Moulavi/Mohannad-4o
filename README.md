# Mohannad 4o - BRAND TRANSLATION + INTERNAL LINK ROOT FIX

این نسخه دو مشکل اصلی را ریشه‌ای اصلاح می‌کند.

## 1) برند انگلیسی

برندهای انگلیسی نباید تبدیل به محصول اشتباه شوند.

مثال غلط:

```text
گز داو
```

مثال درست:

```text
داو
```

برندهای پشتیبانی‌شده:

- Dove → داو
- Cantu → کانتو
- Cliven → کلیون
- Nivea → نیوآ
- Garnier → گارنیر
- L'Oreal → لورآل
- Pantene → پنتن
- Sunsilk → سان‌سیلک
- Vaseline → وازلین
- Bioderma → بایودرما
- CeraVe → سراوی
- Neutrogena → نوتروژینا

## 2) لینک داخلی واقعی و مرتبط

لینک داخلی فقط از `SITE_CATEGORIES` واقعی داخل فایل انتخاب می‌شود.

برای محصولات آرایشی/بهداشتی و مو، این دسته‌ها اولویت دارند:

- شامپو
- مراقبت و زیبایی مو
- مراقبت پوست
- ضد آفتاب
- لوسیون بدن
- بهداشت دهان و دندان
- دئودرانت و ضد تعریق

و این اشتباهات حذف می‌شود:

```text
شامپو → قهوه فوری ❌
کرم مو → نوشیدنی ❌
ماسک مو → گز ❌
```

## قانون جدید

اگر محصول مو/پوست/آرایشی باشد، هیچ لینک خوراکی مثل قهوه فوری، نوشیدنی، گز، سوهان یا زعفران ساخته نمی‌شود.
