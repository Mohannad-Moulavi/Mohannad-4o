# Mohannad 4o - Visible Specs From Image/Text FIX

این نسخه برای جزئیات روی عکس یا ورودی کاربر فیکس شده است.

## قانون جدید

جزئیات زیر اجباری نیستند، اما اگر روی عکس محصول یا در متن کاربر وجود داشته باشند باید حتماً در خروجی بیایند:

- SPF
- PA / PA++ / PA+++
- UVA/UVB
- حجم و وزن مثل 10.4 oz، 300 ml، 250 g
- شماره رنگ / Shade / Color
- رایحه یا طعم
- نوع پوست یا نوع مو
- ترکیبات شاخص روی بسته‌بندی
- مدل، سری، تعداد، رنگ و ویژگی‌های نوشته‌شده روی بسته‌بندی

## محل استفاده

اگر این موارد در عکس یا ورودی مشخص باشند، در این بخش‌ها لحاظ می‌شوند:

- نام اصلاح‌شده محصول
- توضیحات کامل
- بخش مشخصات محصول
- کلیدواژه/عنوان/alt در صورت نیاز

## نکته مهم

این موارد برای همه محصولات اجباری نیستند. اگر روی عکس یا در ورودی وجود نداشته باشند، کد نباید آن‌ها را بسازد.
