# Mohannad 4o - Exact Yoast + Country Series FIX

این نسخه برای سه مشکل اصلی فیکس شده است:

## 1) کشور سازنده
- دیگر متن‌های دستوری مثل «کشور سازنده: اگر مشخص است» وارد خروجی نمی‌شود.
- اگر کشور واقعی مشخص باشد:
  `کشور سازنده: تایلند`
- اگر کشور واقعی مشخص نباشد:
  `کشور سازنده: بستگی به سری ساخت دارد`
- «کشور مبدأ برند» نمایش داده نمی‌شود.

## 2) Yoast SEO
فیلدهای زیر سخت‌گیرانه‌تر اصلاح می‌شوند:
- کلیدواژه کانونی
- عنوان سئو
- توضیحات متا
- توضیحات کوتاه
- متن جایگزین تصویر

قوانین اعمال‌شده:
- کلیدواژه کانونی داخل عنوان، متا، alt و پاراگراف اول توضیحات کامل می‌آید.
- متا حدود 120 تا 155 کاراکتر نگه داشته می‌شود.
- عنوان سئو طبیعی، فروشگاهی و شامل کلیدواژه کانونی است.
- توضیحات کوتاه مخصوص همان دسته محصول تولید می‌شود، نه متن عمومی.

## 3) ساختار Mohannad SEO
- توضیحات کامل کوتاه و ناقص قبول نمی‌شود.
- ساختار HTML با `<p>`, `<hr />`, `<h5>`, `<ul>`, `<li>` حفظ می‌شود.
- بخش کلیدواژه‌ها در یک باکس ترکیبی می‌آید:
  «کلیدواژه‌های مرتبط، مترادف و LSI»
