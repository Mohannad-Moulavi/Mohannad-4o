# Mohannad 4o - Volume Only In Specs FIX

این نسخه طبق اصلاح جدید ساخته شد.

## قانون جدید

- فقط «حجم» نباید در ویژگی‌ها/مزایا تکرار شود.
- حجم فقط در بخش «📦 مشخصات محصول» بیاید.
- اما SPF، PA، UVA/UVB، شماره رنگ، مدل، رایحه، طعم و ویژگی‌های واقعی محصول اگر کاربردی باشند، می‌توانند در ویژگی‌ها هم نوشته شوند.

## مثال درست

```text
ویژگی‌ها:
حاوی ویتامین E و پروویتامین B5
مناسب برای انواع پوست
تغذیه‌کننده و محافظت‌کننده پوست
بافت سبک و جذب سریع

مشخصات:
برند: کلیون (Cliven)
مدل: Crema Polivalente
نوع محصول: کرم چند منظوره
حجم: 300 میلی‌لیتر
کاربرد: تغذیه و محافظت از پوست
```

## مثال برای ضدآفتاب

```text
ویژگی‌ها:
دارای SPF 50
محافظت در برابر UVA/UVB
مناسب استفاده روزانه

مشخصات:
نوع محصول: ضد آفتاب
حجم: 50 میلی‌لیتر
SPF: 50
محافظت: UVA/UVB
```
