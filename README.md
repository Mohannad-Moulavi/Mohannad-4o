# Mohannad 4o - Real Categories + Internal Links + Country Origin FIX

این نسخه طبق لیست دسته‌بندی‌هایی که فرستادی ساخته شد.

## تغییرات اصلی

- 152 دسته‌بندی واقعی سایت داخل کد قرار گرفت.
- لینک داخلی دیگر `#` نیست.
- کد خودش بر اساس محصول بهترین دسته‌بندی واقعی سایت را انتخاب می‌کند.
- مثال:
  - آب زمزم، آب معدنی، نوشیدنی → دسته «نوشیدنی»
  - کافی میت/نسکافه/قهوه فوری → دسته «قهوه فوری»
  - لوسیون بدن → دسته «لوسیون بدن»
  - شامپو → دسته «شامپو»
  - شوینده → دسته «مواد شوینده»
  - آجیل/خشکبار → دسته‌های مربوط به آجیل و خشکبار
- اگر کشور واقعی مشخص باشد، در مشخصات محصول با برچسب «کشور مبدأ» نمایش داده می‌شود.
- اگر کشور مشخص نباشد، نوشته می‌شود:
  `کشور سازنده: بستگی به سری ساخت دارد`
- «کشور مبدأ برند» حذف شده و نمایش داده نمی‌شود.
