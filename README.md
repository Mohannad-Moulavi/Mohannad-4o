# Mohannad 4o - No Duplicate Units FIX

این نسخه مشکل تکرار واحدهای فارسی و انگلیسی را حل می‌کند.

## قانون جدید

اگر یک مقدار هم فارسی و هم انگلیسی/اختصاری آمده باشد، فقط یک‌بار نوشته می‌شود.

مثال:

- `300 میلی‌لیتر 300 ml` ❌
- `300 میلی‌لیتر` ✅

- `250 گرم 250 g` ❌
- `250 گرم` ✅

## نکته مهم

- اگر فقط `10.4 oz` روی عکس باشد، همان حفظ می‌شود.
- اگر فقط `300 ml` روی عکس باشد و فارسی‌اش نیامده باشد، همان حفظ می‌شود.
- اگر هر دو با یک عدد کنار هم باشند، تکراری حذف می‌شود و واحد فارسی ترجیح داده می‌شود.
- مواردی مثل SPF، PA و UVA/UVB هم اگر دوبار تکرار شوند، یک‌بار نمایش داده می‌شوند.

## امکانات حفظ‌شده

- کشور کلاً حذف است.
- لینک داخلی واقعی از دسته‌بندی‌های سایت noon-valqalam.ir
- تشخیص محصول از عکس مهم‌تر از اسم تایپی
- اصلاح اسم محصول
- جزئیات عکس مثل SPF، oz، ml، شماره رنگ و رایحه اگر وجود داشته باشد ثبت می‌شود.

- قوانین Yoast SEO
