# Mohannad 4o - REBUILD SPECS ROOT FIX

این نسخه مشکل را از ریشه حل می‌کند.

## علت واقعی

مشکل این بود که مشخصات محصول از خروجی مدل قبول می‌شد و بعد چند مرحله پاکسازی روی آن انجام می‌شد. اما مدل گاهی مشخصات را چندبار، با HTML ناقص یا متن ساده تولید می‌کرد؛ برای همین پاکسازی خطی کافی نبود.

## راه‌حل جدید

دیگر بخش «📦 مشخصات محصول» از مدل قبول نمی‌شود.

کد این کارها را انجام می‌دهد:

1. تمام بخش‌های مشخصات تولیدشده توسط مدل را حذف می‌کند.
2. مشخصات معتبر را از متن، عکس/ورودی و خروجی مدل استخراج می‌کند.
3. مشخصات را خودش دوباره با ساختار استاندارد می‌سازد.
4. برای «حجم» فقط یک خط نگه می‌دارد.
5. اگر حجم فارسی و انگلیسی هر دو باشد، فارسی را نگه می‌دارد.

## نتیجه درست

```text
📦 مشخصات محصول:
برند: کلیون
مدل: Polivalente
نوع محصول: کرم چندمنظوره
حجم: 300 میلی‌لیتر
کاربرد: تغذیه و محافظت از پوست
```

دیگر این نباید اتفاق بیفتد:

```text
حجم: 300 میلی‌لیتر
حجم: 300 ml
```
