# Mohannad 4o - Focus Keyword Product + Brand FIX

این نسخه مشکل کلیدواژه ناقص را حل می‌کند.

## مشکل

برای محصولی مثل:

```text
کرم نرم‌کننده مو بدون نیاز به آبکشی Cantu حاوی شی باتر وزن 340 گرم
```

نسخه قبلی کلیدواژه را این‌طور می‌ساخت:

```text
کرم نرم‌کننده مو بدون
```

این غلط است، چون «بدون» ویژگی ناقص است و برند محصول حذف شده.

## قانون جدید

کلیدواژه کانونی باید از این فرمول ساخته شود:

```text
نوع محصول + برند
```

## خروجی درست برای مثال بالا

```text
کرم نرم‌کننده مو کانتو
```

یا در صورت نیاز کوتاه‌تر:

```text
کرم مو کانتو
```

## مواردی که نباید وارد کلیدواژه شوند

- بدون نیاز به آبکشی
- حاوی شی باتر
- وزن 340 گرم
- حجم
- مدل
- تعداد
- SPF و PA مگر خود محصول با همین عبارت جستجو شود

این موارد باید در توضیحات، مشخصات و ویژگی‌ها بیایند، نه اینکه کلیدواژه کانونی را ناقص کنند.

## امکانات حفظ‌شده

- قوانین حرفه‌ای Yoast
- کلیدواژه در پاراگراف اول
- کلیدواژه در زیرعنوان
- کلیدواژه در متا، عنوان و alt
- توضیحات متا ۱۲۰ تا ۱۵۵ کاراکتر
- حجم فقط یک‌بار در مشخصات
- مشخصات محصول از نو ساخته می‌شود
- کشور حذف است
