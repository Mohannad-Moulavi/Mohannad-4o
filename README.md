# Mohannad 4o - SAMPLE STYLE YOAST FOCUS FIX

طبق نمونه سالمی که فرستادی، کلیدواژه فقط کوتاه‌سازی کورکورانه نیست.

## قانون جدید

کلیدواژه کانونی حرفه‌ای:

```text
نوع محصول + برند + ماده شاخص
```

## مثال

برای:

```text
ماسک مو عمیق Cantu حاوی Shea Butter برای موهای فر و مجعد
```

خروجی درست:

```text
ماسک مو کانتو شی باتر
```

عنوان سئو:

```text
خرید ماسک مو کانتو شی باتر | تقویت‌کننده موهای فر و مجعد
```

متن جایگزین تصویر:

```text
ماسک مو کانتو شی باتر برای موهای فر و مجعد
```

## مشکل رفع‌شده

دیگر این کلیدواژه ناقص تولید نمی‌شود:

```text
کرم نرم‌کننده مو بدون
```

برای Cantu Leave-In خروجی بهتر:

```text
کرم نرم‌کننده مو کانتو
```

یا اگر شی باتر هویت محصول باشد:

```text
کرم مو کانتو شی باتر
```
