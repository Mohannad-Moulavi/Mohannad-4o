# Mohannad 4o - Image First + Name Correction FIX

این نسخه برای مشکل «اسم الکی + عکس محصول واقعی» فیکس شده است.

## قانون جدید

- اگر تصویر محصول ارسال شده باشد، تصویر و متن روی بسته‌بندی منبع اصلی است.
- نام خامی که کاربر تایپ کرده فقط راهنماست و ممکن است غلط یا الکی باشد.
- correctedProductName باید نام اصلاح‌شده واقعی محصول باشد، نه تکرار کورکورانه نام کاربر.
- اگر کاربر اسم الکی بزند ولی عکس آب زمزم بفرستد، خروجی باید آب زمزم / نوشیدنی / مواد غذایی شود، نه آرایشی.
- گارد قبلی که اسم خام کاربر را دوباره به خروجی برمی‌گرداند، برای حالت دارای عکس غیرفعال شده است.
- تشخیص دسته برای آب زمزم، آب معدنی و نوشیدنی به سمت مواد غذایی/نوشیدنی هدایت شده است.

## مدل پیشنهادی برای کیفیت

```env
GITHUB_MODEL=openai/gpt-4o
MAX_OUTPUT_TOKENS=4000
AI_MODEL_TIMEOUT_MS=60000
WEB_SEARCH_TOTAL_TIMEOUT_MS=24000
WEB_SEARCH_TIMEOUT_MS=10000
```

برای تشخیص عکس محصول، مدل ضعیف مثل gpt-4o-mini مناسب نیست. این نسخه پیش‌فرض را روی GPT-4o کامل نگه می‌دارد.
